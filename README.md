# CPG Retail Sales Performance Analysis

## Project Overview
This project demonstrates an end-to-end data pipeline, transforming a messy dataset of 3,000+ transactions into a high-impact sales dashboard. It showcases the ability to handle real-world data challenges, from raw data extraction and Power Query cleaning to advanced DAX modeling and business forecasting.

## Tech Stack & Skills
*   **Data Cleaning:** Microsoft Excel & Power Query.
*   **Data Modeling:** Excel Data Model (Star Schema).
*   **Analytics:** DAX (Time Intelligence & Profitability Measures).
*   **Soft Skills:** Business Problem Solving & Data Integrity.

## Business Value & Key Objectives
This analysis was designed to provide management with actionable answers to:
*   **Growth:** Identifying regions and products seeing Year-over-Year (YoY) growth.
*   **Profitability:** Analyzing how discount rates impact actual profit margins versus sales volume.
*   **Customer Insights:** Ranking top customers by revenue and understanding purchase frequency.
*   **Forecasting:** Identifying seasonal patterns to project sales for upcoming quarters.

## Data Cleaning Deep-Dive (The Process)
Real-world data is rarely ready for analysis. In this project, I addressed:
*   **The Date Challenge:** Implemented custom Power Query logic to standardize mixed date formats using `Date.FromText`.
*   **Data Standardization:** Fixed inconsistent product names, customer segments, and regional labels.
*   **Integrity Checks:** Performed deduplication and null-value handling to ensure a 100% accurate dataset for modeling.
*   **Relationship Building:** Created a Star Schema connecting Sales, Product, Customer, and Calendar tables.

## Key Metrics Created (DAX)
I developed a suite of automated measures to track business health:
*   **Total Sales & Profit Margin %**[cite: 1].
*   **YoY Growth:** Comparing performance against the same period in the previous year.
*   **YTD/MTD Tracking:** Providing real-time progress against monthly and annual targets.
*   **Discount Impact:** Measuring revenue loss vs. volume gain from high discount rates.
